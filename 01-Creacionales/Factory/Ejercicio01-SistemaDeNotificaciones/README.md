# Ejercicio 01 - Sistema de Notificaciones
## Patrón utilizado
Factory Method
## Historia de usuario
Como usaurio del sistema, quiero recibir notificaciones por diferentes medios,
como correo, SMS o WhatsApp, para estar informado según el canal que elija.
## Solicitud 
Crea un sisma donde el usuario pueda elegir el tipo de notificacion: "email" ,
"sms" o "whatsapp". Según esa opción, el programa debe crear el objeto crrespondiente
usando una Factory
